Publications:
• A.Hosseini, Z.Yahouni. M.Feizabadi “Scheduling AIV Transporter Using Simulation-Based
Supervised Learning: A Case Study on a Dynamic Job-Shop with Three Workstations” _
Published in the IFAC World Congress 2023
• Dynamic job shop scheduling is a problem that consists of scheduling jobs dynamically
with different routing on a set of machines. A feasible and quick solution can be
computed using heuristics. One well-known heuristic in job shop problems is selecting
the priority dispatching rule (such as giving priority to the job with the Shortest Processing
Time called SPT). Nowadays, with the application of industry 4.0 technologies such as
sensors, Intelligent robots, etc., workshop data are more accessible and can be exploited
to find the appropriate dispatching rule depending on the state of the shop. This work
proposes a data-driven methodology for scheduling an AIV (Autonomous Intelligent
Vehicle) that supplies three workstations. Our approach is based on data collected from
an Arena simulation model fed to a supervised learning algorithm. This one helps identify
one among five dispatching rules for each scheduling decision.
• A.Hosseini, Z.Yahouni. M.Feizabadi “Resolving Flexible Job Shop Scheduling with multiple AIVs
transporters using Multi-Agent Deep Reinforcement Learning algorithms” _ In progress
• Flexible Job Shop Scheduling is a NP-hard problem that consists of scheduling Jobs with
different rooting, which means different operations to be executed, in addition to the
multiple choices of machines for each operation. Furthermore, Multiple choices of AIV
transporters with specific capacity make the problem more complex. Various single-agent
deep reinforcement learning methods including Q learning, PPO, A2C etc. have been
applied in such cases. However, recent research showed that Multi Agent Architectures
outperform approached where a single DRL agent makes all the decisions. In this work,
we propose a multi agent cooperative architecture to deal with the aforementioned
complexities and the high dimensionality of state space and action space.
• M.Feizabadi “A new approach of cognitive robot using theories of consciousness” _in progress
• In psychology, cognition refers to the acquiring knowledge and understanding through the
senses, thoughts and experiences. The definition of cognition has been expanded in the
field of artificial intelligence. In other words, in this area, cognition includes not only the
above concepts, but also the process of creating motions by considering learnings,
memories and motion control signals. On the other hand, several theories of
consciousness have been proposed in recent years. However, it is unclear how current
theories relate to each other or weather be distinguished from each other. In this work,
inspired by the biological status of memory and neurons in addition to a combination of
two prominent theories of consciousness, Global Workspace Theory and Integrated
Information Theory, we propose a new approach to deal with the artificial consciousness
of a robot by combining two prominent theories.
